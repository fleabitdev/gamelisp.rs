Alphabetic Index
(let-mixin name ..clauses)
  name: a sym
  clauses: zero or more forms
  returns #n
M

Defines a mixin and binds it to a local variable.

let-mixin is equivalent to let-class, except that it calls the mixin macro rather than the class macro.
