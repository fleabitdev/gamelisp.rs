(finish-block name (? result #n))
  name: a sym
  result: any type (optional)
S

Jumps to the end of an enclosing block.

There must be an enclosing block form with the given name. When the finish-block form is evaluated, execution jumps ahead to the end of that block. The block as a whole evaluates to result.

(prn (block example
  (finish-block example 100))) ; prints 100
