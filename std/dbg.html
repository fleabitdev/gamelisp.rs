(dbg ..args)
  args: zero or more arguments (any type)
M

Prints its arguments, formatted for debugging.

Prints each argument on its own line using eprn, prefixed with its file location and the argument's form. For example, (dbg (+ 1 2) (+ 3 4)) might print:

[example.glsp:1] (+ 1 2) = 3
[example.glsp:1] (+ 3 4) = 7
