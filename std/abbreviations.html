Abbreviations

Some forms are used so frequently that writing out the full (callee arg) syntax every time would be very inconvenient.

For those forms, we provide abbreviations: alternative syntax for a particular form, intended to be easier to read and more convenient to type.

For example, wherever you would write (access coll key), you can write [coll key] instead. When it comes to parsing and printing, the two pieces of text are completely interchangeable.

'

The abbreviation for quote.

`

The abbreviation for backquote.

~

The abbreviation for unquote.

..

The abbreviation for splay.

@

The abbreviation for atsign.

.

The abbreviation for meth-name.

[]

The abbreviation for access.

"{}"

The abbreviation for template-str.
